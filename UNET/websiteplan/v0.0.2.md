# MVP Prompt v0.0.2 - Scale from Ground Version

## Goal

Scale dari `v0.0.1` menjadi workflow semi-annotation tool yang siap dipakai membuat pseudo ground-truth dengan produktivitas lebih tinggi.

## Dependency

Wajib sudah stabil di `v0.0.1`:

- single upload
- inferensi berhasil
- review mask black-and-white

## New Features (Incremental)

1. Multi-image upload (batch kecil dulu, mis. 20-50).
2. Batch processing progress bar (`queued`, `processing`, `done`, `failed`).
3. Mask editor dasar:
   - brush
   - eraser
   - undo/redo
4. Smart selection (click-assisted), bisa berbasis:
   - SAM endpoint, atau
   - magic-wand sederhana (color similarity) sebagai fallback.
5. Save edited mask per image.
6. Export hasil:
   - `images/`
   - `masks/`
   - `metadata.csv`

## Extended API (Target)

- `POST /predict-batch`
- `GET /jobs/{job_id}`
- `POST /mask/edit`
- `POST /smart-select`
- `GET /export/{job_id}`

## Data Contract

- Original image dipertahankan.
- Mask final wajib binary (`0/255`).
- Metadata minimal:
  - `image_name`
  - `mask_name`
  - `source` (`auto`/`edited`)
  - `created_at`

## UX Requirements

- Side-by-side viewer tetap dipertahankan.
- Navigasi antar image cepat (next/prev).
- Tool state jelas (brush aktif, ukuran brush, threshold).
- Ada warning jika user pindah image tanpa save.

## Risk Focus di v0.0.2

1. Label noise dari auto mask.
2. Performa browser saat banyak image resolusi besar.
3. Inkonsistensi mask jika editor tidak lock ke binary mode.
4. Ekspektasi smart selection terlalu tinggi.

## Mitigations

1. Tambahkan quick QA workflow (`needs_review`, `approved`).
2. Limit resolusi preview + simpan full-res di backend.
3. Enforce binary conversion saat save.
4. Sediakan fallback manual brush jika smart-select gagal.

## Acceptance Criteria v0.0.2

- Batch upload dan processing berjalan stabil.
- User bisa edit dan save mask.
- Export dataset berhasil dan terbaca untuk training pipeline.
- Tidak ada mask non-biner di hasil export.

## Out of Scope (Keep for Later)

- Multi-user collaboration.
- Role/permission.
- Active learning retraining loop otomatis.
- Cloud autoscaling production setup.

## Definition of Done

Versi `v0.0.2` selesai jika tim bisa:

- upload banyak image,
- auto-generate mask,
- review/edit,
- export dataset siap train secara repeatable.
